**Goal:** Publish your first doc in a few minutes using the uDos **Publishing** student path.

## Before you begin

- You write in **Markdown** (we teach **GFM** — see [GFM guide](gfm-guide.html)).
- **Premium** unlocks one-click pipelines, Marp, MDX, full theming, and OAuth “brown paper bag” deploy — earned via contribution or GitHub pathway, not only payment ([locked model](https://github.com/fredporter/uDosDev/blob/main/docs/specs/v4/UDOS_VALUE_LADDER_AND_PUBLISHING_PREMIUM_v1.md)).

## Five-minute path (illustrative)

1. Create or open a Markdown project (book, site, slides).
2. Run the student publish command when your tooling ships it (see [Brown paper bag](https://github.com/fredporter/uDosDev/blob/main/docs/specs/v4/UDOS_VALUE_LADDER_AND_PUBLISHING_PREMIUM_v1.md#4-brown-paper-bag-packaging--locked-spec) in the spec — `udos publish` is the **intent**, exact CLI ships in product repos).
3. Connect **GitHub Pages** (and OAuth where Premium applies) per [One-click deploy](premium/one-click-deploy.html).
