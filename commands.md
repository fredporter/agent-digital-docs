# Commands

## Agent Digital

- **RESEARCH** — gather information and sources
- **BINDER** — structure a project
- **REFINE** — improve existing structure
- **COMPILE** — generate final project files
- **STATUS** — show current completeness
- **LOCK** — freeze scope and stop new assumptions

## uDOS Developer

- **SCAN** — inspect project or repository context
- **BRIEF** — create a code binder
- **REFINE** — improve structure or clarity
- **PATCH** — propose structured changes
- **EXPORT** — prepare for IDE or tool handoff
- **RELEASE** — prepare a public release
- **STATUS** — check current completeness
