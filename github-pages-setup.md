# GitHub Pages Setup

## 1. Create the repository

Example:
`agentdigital-docs`

## 2. Upload the docs

Upload the contents of the `docs/` folder from this pack.

## 3. Enable GitHub Pages

In GitHub:

1. Open **Settings**
2. Open **Pages**
3. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/docs`

## 4. Save

GitHub will publish your site to a URL like:

`https://fredporter.github.io/agentdigital-docs/`

## Notes

- Keep public docs practical and public-safe
- Do not upload hidden GPT knowledge files into the docs repo
- Keep the docs simple and useful
