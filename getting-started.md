# Getting Started

## Agent Digital

### 1. Start with an idea
Describe what you are trying to explore, organise, or build.

### 2. Define the scope
Use normal conversation to clarify:
- what the project is
- what matters most
- what is still unclear

### 3. Use commands when ready
- **RESEARCH** — gather structured information
- **BINDER** — turn the idea into a project binder
- **REFINE** — improve structure or clarity
- **COMPILE** — generate final files

### 4. Keep the output
The result is designed to be portable and usable outside chat.

## uDOS Developer

Use uDOS Developer when a project needs code, repositories, or implementation planning.

Start by selecting one path:

- **FAMILY** — build a new project
- **FORK** — adapt existing work
- **CONTRIBUTE** — improve an existing project

Then use:
- **SCAN** — inspect context
- **BRIEF** — create a code binder
- **EXPORT** — prepare handoff to tools
- **RELEASE** — prepare a public release
