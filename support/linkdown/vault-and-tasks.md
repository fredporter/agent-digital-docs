**Scope — Linkdown Core (free):** Vault layout, Markdown files, Task Forge checklists, and the local index are **Core**. Anything that depends on the **premium MCP runtime** is covered under [Premium & MCP](premium-mcp.html).

## Vault

Your **vault** is a folder on disk. Linkdown keeps **Markdown as the source of truth**; optional derived data (search index, link graph) lives under `.linkdown/` inside that folder.

- **Master vault mode** — When enabled, paths such as `Binders/` and `Data/` align with common Obsidian-style and Agent Digital binder conventions.
- **Quick Open & search** — Use the local index for fast title/path search and full-text search where indexing is available.

## Task Forge & checklists

Linkdown supports **GFM+Task**: checkbox lists (`- [ ]` / `- [x]`) compatible with Task Forge and list-oriented notes. See the core documentation for [Task Forge](https://github.com/fredporter/Linkdown-core/blob/main/docs/TASK_FORGE.md) and [Obsidian compatibility](https://github.com/fredporter/Linkdown-core/blob/main/docs/OBSIDIAN_AND_TASK_FORGE.md).

## Lists

With **Lists** enabled in Settings, you can work with dedicated list notes and templates (see core `docs/guide/`).
