**Scope — Linkdown Core (free):** The desktop UI and preview described here ship in the **public** AGPL build. Optional Premium-only panels or tools are noted on [Premium & MCP](premium-mcp.html).

Linkdown’s interface follows the **Inkdown** lineage: a **sidebar** (folders / vault tree), a **rich editing surface** for Markdown, and a **preview** or split layout for rendered output.

Typical areas:

- **Sidebar** — Vault folders, binders, and note hierarchy. Context menus create folders, notes, and (when enabled) list notes.
- **Editor** — WYSIWYG-style editing with valid GFM output; code fences, tables, and task lists behave like GitHub.
- **Preview** — Renders Markdown with extensions (e.g. Shiki-highlighted code blocks in recent versions, Mermaid diagrams when enabled).
- **Outline** — Optional heading outline for long documents (Settings).
- **Chat** — Optional side panel when LLM features are enabled.

The **desktop screenshots** on the [support home](index.html) show the classic Inkdown layout that Linkdown continues to ship.
