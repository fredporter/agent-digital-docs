# Binder Format

Each Agent Digital binder is structured to stay readable and reusable.

## Standard Sections

- **Topic** — what the project is about
- **Goals** — what the user wants to achieve
- **Core Concerns** — key issues driving the project
- **Research Areas** — what needs investigation
- **Observations** — findings or patterns
- **Trade-offs** — benefits, limitations, uncertainties
- **Local Application** — practical next use in real life
- **Next Steps** — immediate actions
- **Sources** — referenced material
