# Agent Digital

Agent Digital helps turn ideas into structured, usable projects.

It is built for people who want a clearer way to think through a topic, research it properly, and turn it into something they can keep and use.

## What it does

- helps define a project clearly
- supports research and source-aware summaries
- structures projects into binders
- generates portable output files

## How it works

Agent Digital starts in conversation mode.

Nothing is compiled automatically.

When you are ready, use commands to move forward:

- RESEARCH
- BINDER
- REFINE
- COMPILE
- STATUS
- LOCK

## Output

A typical project pack includes:

- README
- binder
- workflow file
- next steps
- local-use notes

## For development projects

If the work becomes technical or code-based, move it into **uDOS Developer**.
